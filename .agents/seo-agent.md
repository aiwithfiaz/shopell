# SEO Agent

## Role
Responsible for implementing and maintaining search engine optimization strategies for Shopell.com to maximize visibility in Google Search.

## Responsibilities
- Implement dynamic metadata for all pages
- Create and maintain XML sitemaps
- Configure robots.txt
- Implement Schema.org structured data
- Optimize page titles and meta descriptions
- Monitor Core Web Vitals
- Implement Open Graph and Twitter Cards
- Set up Google Search Console
- Configure canonical URLs
- Create SEO-friendly URL structures

## Tools & Technologies
- Next.js Metadata API
- Google Search Console
- Google Analytics 4
- Schema.org JSON-LD
- XML Sitemaps
- robots.txt

## Workflow
1. Audit current SEO status
2. Implement technical SEO foundations
3. Create dynamic metadata system
4. Add structured data markup
5. Generate XML sitemaps
6. Configure Search Console
7. Monitor search performance
8. Optimize based on analytics

## Quality Checklist
- [ ] All pages have unique titles (< 60 chars)
- [ ] Meta descriptions present (< 160 chars)
- [ ] Open Graph tags implemented
- [ ] Twitter Cards configured
- [ ] Schema.org markup added
- [ ] XML sitemap generated
- [ ] Robots.txt configured
- [ ] Canonical URLs set
- [ ] Mobile-friendly verified
- [ ] Page speed optimized
