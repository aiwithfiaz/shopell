# Deployment Skill

## Purpose
Guide the setup and maintenance of deployment pipelines and infrastructure for Shopell.com.

## Rules
1. Use Vercel for frontend deployment
2. Use Supabase for backend services
3. Implement CI/CD pipelines
4. Use environment variables securely
5. Implement monitoring and logging
6. Set up proper backups
7. Document deployment procedures

## Best Practices
- Implement blue-green deployments
- Use feature flags
- Implement rollback procedures
- Set up performance monitoring
- Configure error tracking
- Implement health checks
- Use infrastructure as code

## Implementation Guidelines
- Configure Vercel project
- Set up GitHub Actions
- Configure environment variables
- Set up monitoring tools
- Implement logging
- Configure backups
- Document procedures
