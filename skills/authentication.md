# Authentication Skill

## Purpose
Guide the implementation of secure, user-friendly authentication and authorization systems for Shopell.com.

## Rules
1. Use Supabase Auth for authentication
2. Implement secure password hashing (bcrypt)
3. Use JWT tokens for sessions
4. Implement proper CSRF protection
5. Use secure HTTP-only cookies
6. Implement rate limiting
7. Follow OWASP security guidelines

## Best Practices
- Implement multi-factor authentication
- Use secure password reset flows
- Implement proper session management
- Use role-based access control (RBAC)
- Implement audit logging
- Use secure OAuth flows
- Implement proper logout procedures

## Implementation Guidelines
- Configure Supabase Auth providers
- Implement login/register flows
- Create protected route middleware
- Implement role-based access
- Create user profile management
- Implement password reset flows
- Set up email verification
