# Frontend Development Skill

## Purpose
Guide the development of high-quality frontend components and pages for Shopell.com using modern React patterns and best practices.

## Rules
1. Always use TypeScript for type safety
2. Follow React Server Components patterns when possible
3. Use Client Components only when interactivity is required
4. Implement proper loading and error states
5. Ensure mobile-first responsive design
6. Follow Shadcn UI component patterns
7. Use Tailwind CSS for styling consistently

## Best Practices
- Use semantic HTML elements
- Implement proper ARIA attributes
- Optimize images with next/image
- Use dynamic imports for heavy components
- Implement proper key props for lists
- Use React.memo for expensive renders
- Implement proper cleanup in useEffect

## Implementation Guidelines
- Create components in `/src/components/`
- Use barrel exports (index.ts)
- Follow naming conventions: PascalCase for components
- Keep components small and focused
- Extract reusable logic into custom hooks
- Use proper TypeScript interfaces
