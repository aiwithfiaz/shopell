# SEO Optimization Skill

## Purpose
Guide the implementation of comprehensive SEO strategies for Shopell.com to maximize search engine visibility.

## Rules
1. Implement unique meta titles (< 60 chars)
2. Create compelling meta descriptions (< 160 chars)
3. Use proper heading hierarchy (H1-H6)
4. Implement Schema.org structured data
5. Create SEO-friendly URLs
6. Optimize images with proper alt text
7. Implement canonical URLs

## Best Practices
- Use Open Graph tags for social sharing
- Implement Twitter Cards
- Create XML sitemaps
- Configure robots.txt
- Implement breadcrumbs
- Use internal linking strategies
- Monitor Core Web Vitals

## Implementation Guidelines
- Use Next.js Metadata API
- Implement dynamic metadata
- Create structured data components
- Generate XML sitemaps
- Configure Search Console
- Set up Google Analytics
- Monitor search performance
