# Security Skill

## Purpose
Guide the implementation of comprehensive security measures for Shopell.com to protect user data and prevent vulnerabilities.

## Rules
1. Validate all user inputs with Zod
2. Implement CSRF protection
3. Use secure HTTP headers
4. Implement rate limiting
5. Prevent SQL injection
6. Implement XSS protection
7. Use secure authentication

## Best Practices
- Implement Content Security Policy
- Use secure cookie settings
- Implement proper error handling
- Use HTTPS everywhere
- Implement API key rotation
- Use secure file upload validation
- Implement audit logging

## Implementation Guidelines
- Configure security headers
- Implement input validation
- Set up rate limiting
- Configure CORS properly
- Implement secure file uploads
- Set up error monitoring
- Conduct security audits
