# Tailwind Design Skill

## Purpose
Guide the creation of consistent, beautiful, and responsive UI designs for Shopell.com using Tailwind CSS.

## Rules
1. Use the design system tokens (colors, spacing, typography)
2. Follow mobile-first approach
3. Use CSS variables for theming
4. Implement dark/light mode consistently
5. Use responsive breakpoints appropriately
6. Follow Shadcn UI component patterns
7. Keep styles maintainable and organized

## Best Practices
- Use @apply for repeated patterns
- Implement proper focus states
- Use transition utilities for animations
- Implement proper hover and active states
- Use container utilities for layouts
- Implement grid system consistently
- Use proper spacing scale

## Implementation Guidelines
- Define custom colors in globals.css
- Use CSS variables for theme colors
- Implement consistent border radius
- Use proper font sizes and weights
- Implement consistent shadows
- Use proper z-index scale
- Implement responsive typography
