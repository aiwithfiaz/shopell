# Testing Skill

## Purpose
Guide the implementation of comprehensive testing strategies for Shopell.com to ensure code quality and reliability.

## Rules
1. Write tests for all critical paths
2. Use appropriate testing frameworks
3. Maintain high test coverage
4. Write deterministic tests
5. Mock external dependencies
6. Test edge cases
7. Automate test execution

## Best Practices
- Use testing pyramid approach
- Write unit tests for utilities
- Create component tests
- Implement integration tests
- Set up E2E tests
- Use test-driven development
- Maintain test documentation

## Implementation Guidelines
- Configure Jest for unit tests
- Set up React Testing Library
- Configure Playwright for E2E
- Create test fixtures
- Set up CI/CD test runs
- Monitor test coverage
- Document testing procedures
